# marmi_app

Um Projeto para Gerenciamento de Marmitarias

## Getting Started

Este projeto foi desenvolvido por mim como forma de aplicar os estudos de flutter.

A ideia foi partir de uma demanda real e tentar propor algum tipo de solução que empregasse as novas tecnologias que aprendi.

Embora pareça simples, uma Marmitaria tem vários clientes, cada cliente pode encomendar marmitas para diferentes dias da semana, e cada um pode ter uma forma de pagamento diferente.
O App pretende fornecer ao usuário uma visão de quem são os seus clientes, qual a programação de entregas do dia, e em foma de calendário saber os dias que deve entregar para aquele cliente, bem como os dias que já foram pagos.


## Principais Recursos

- Armazenamento em nuvem (Firebase)
- Login 
- Cadastro dos clientes
- Lançamento da programação de entregas do cliente
- Cadastro dos produtos
- Movimentação Financeira 
- Relatórios
